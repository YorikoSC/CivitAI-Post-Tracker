# Dashboard Guide

This dashboard is a local analytics view generated by the tracker.

It is based on collected historical snapshots, not just the current state of a post.

## What the dashboard shows

### Tracked posts

The number of posts included in tracking after the configured start point.

### Known totals / Unknown totals

Shows whether reaction totals were successfully collected for posts.

### Reactions today by type

A summary of reaction growth during the current local day, split by type:

- Like
- Heart
- Laugh
- Cry

This is based on **daily reaction delta**, not on lifetime totals.

### Best post today

The best-performing post for the current local day based on total reaction gain.

The block also shows the reaction breakdown by type.

### Best post this week

The best-performing post for the last 7 days based on reaction gain during that period.

This is not based on lifetime totals.

### Suggested posting windows

Suggested posting time windows derived from historical performance.

These are advisory hints, not strict rules.

A strong post can outperform a weak post even in a worse time slot.

### Post list

The post table usually includes:

- Post ID / link
- Published time
- Total reactions
- Comments
- Last seen / last update

## Important interpretation notes

### Suggested posting windows are not absolute

They are based on historical data and may be unreliable when:

- the sample size is small
- content types vary too much
- one unusually strong post skews the result
- platform behavior changes

### Best post today / week uses period gain

These blocks should be interpreted as:

- reaction gain during the current local day
- reaction gain during the last 7 days

They are not simply the oldest post with the highest lifetime total.

### Timezone matters

Dashboard periods are interpreted using the configured local timezone.

## Practical use

Use the dashboard to answer questions like:

- Which recent posts gained reactions fastest?
- What time slots produced the best first-day performance?
- Which post is currently performing best this week?
- How much engagement came in today, and of what type?

## What the dashboard does not guarantee

- It does not predict future performance with certainty.
- It does not know content quality.
- It does not replace your own judgment.
- It should be treated as a decision aid, not an autopilot.
