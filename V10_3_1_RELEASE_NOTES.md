# CivitAI Tracker v10.3.1

## Hotfix

This small maintenance update cleans up Dashboard Collections activity tables.

## Fixed

- Simplified Collections activity rows in the Dashboard.
- Removed a low-value event metadata column from collection activity tables.
- Kept collection image links, previews, period filters, and post mapping status unchanged.

## Updating

EXE users can update through **Updates** once the portable package is available.

Source-mode users should update through Git.

Expected portable package:

```text
CivitAITracker-v10.3.1-win64.zip
```

If GitHub Release assets are unavailable on your network, use the mirror package link from the release notes or download the ZIP manually and choose **Select ZIP** in the Updates dialog.

Mirror line format:

```text
Update package mirror: https://example.com/CivitAITracker-v10.3.1-win64.zip
```
