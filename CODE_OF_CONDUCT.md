# Code Of Conduct

This project should stay useful, calm, and welcoming for people reporting bugs, asking questions, or contributing fixes.

## Expected Behavior

- Be respectful and direct.
- Assume good intent when a report is incomplete.
- Keep technical criticism focused on the code, behavior, or documentation.
- Respect privacy. Do not ask people to publish API keys, private databases, or unredacted logs.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Publishing someone else's private information.
- Repeated off-topic or disruptive comments.
- Pressuring users to share sensitive files publicly.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct. Repeated or severe violations may result in blocked participation.

Security-sensitive problems should be reported through `SECURITY.md`.
